Navigating the Program Pages

Detailed descriptions of the contents of different program pages are available in the "Program Pages" section of the Help menu. Program tabs are arranged hierarchically to lead users through the basic steps involved in conducting various types of analyses. The main set of tabs along the top of the program box includes the following:

  • Getting Started contains a title page with brief overview of the program and its various pages.
  • Rigorous Rigid-Block Analysis contains a series of pages to lead users through a rigorous rigid-block Newmark analysis.
  • Simplified Analyses contains pages that allow users to conduct (1) various types of simplified rigid-block analyses or (2) simplified decoupled analysis.
  • Record Manager contains pages allowing users to add earthquake records to the database and to modify their properties.
  • Utilities contains programs designed to convert existing strong-motion files into different forms.
  • Help contains help files for each of the program pages and detailed documentation of Newmark's method.
